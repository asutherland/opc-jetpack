Planet Ubiquity Redesign Documentation

This is the documentation for an experimental redesign of Planet Ubiquity, but the underlying code can easily be modified to serve as a dashboard for any kind of project. In fact, it's currently being repurposed as a dashboard for the Jetpack project.

For more information on the rationale behind this experiment, see Atul's blog post entitled Redesigning Planets and Project Dashboards.
