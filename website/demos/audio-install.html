Audio Recording Demo

Around 100 lines of code to implement a voice memo application that:
1) Adds a Record/Stop button to the status bar.
2) Adds a 'Voice Memos' slideBar that will be activated whenever the user visits a page on which there is a voice memo, and can also be used to play back any of the recorded memos.
3) Uses Simple Storage to keep track of recorded audio files, the time at which they were recorded and the web page they are for.
