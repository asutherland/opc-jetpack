UnAd - Content Filter Demo

Demonstrates in ~80 lines of code how to build a Jetpack feature that can remove unwanted scripts, images, and iframes with a pretty UI for turning the feature on and off. It uses a blocklist created for AdBlock Plus.

Of course, you'll need to install Jetpack if you haven't already.
