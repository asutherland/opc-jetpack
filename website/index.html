Jetpack is a Mozilla Labs project to explore new ways to extend & personalize the Web.

The add-ons community for Firefox is arguably one of the largest, most vibrant sources for innovation on the Web today. However, we've only scratched the surface of its potential.

Jetpack is an experiment in using open Web technologies to enhance the browser, allowing anyone who can build a Web site to participate in making the Web a better place to work, communicate and play. Jetpack also aims to allow new features to be added to the browser without a restart or compatibility issues, resulting in little to no disruption to the online experience.

As with all Mozilla Labs initiatives, Jetpack is an open source project and everyone is welcome to participate in its design, development & experimentation.

Guiding Principles

Web-based

  • HTML, Javascript, and CSS will be the only tools required
  • Full AJAX & multimedia capabilities of modern Web browsers, including <canvas> and <video support
  • Streamed & accessible via a URL, just like the rest of the Web
  • Debugged in the browser without restarting using common Web development tools like Firebug

Secure

  • Provides access to only the privileges needed, with security issues always be presented in human-terms, and not technical-terms
  • Easy to review code ensures that potential security issues are shallower, and review times shorter

Robust

  • Versioned APIs so that you won't have to update & revalidate your code for every new version of Firefox
  • Lightweight but full featured, enabling simple and complex applications that add to the Web experience, just like Firefox add-ons do today

Extensible

  • Archiecture will allow for the easy inclusion of reviewed & versioned third-party toolkits (e.g. jQuery, Dojo, etc.) and API libraries (e.g. Twitter, Delicious, Google Maps, etc.)
